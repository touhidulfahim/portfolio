# Mohammad Touhidul Islam Portfolio Website

A modern responsive portfolio website created from the provided CV and photo.

## Files
- `index.html` - main website markup
- `styles.css` - responsive styling and visual design
- `script.js` - mobile menu, active navigation, and scroll reveal animation
- `assets/profile.jpg` - optimized profile image
- `assets/Mohammad-Touhidul-Islam-CV.pdf` - downloadable CV

## How to run
Open `index.html` in any browser.

## How to deploy free
You can deploy this folder to GitHub Pages, Netlify, Vercel, or Cloudflare Pages.
